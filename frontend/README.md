# Frontend
>Quasar Progressive Web App 

### Installation
```sh
$ cd frontend
$ npm install 
$ quasar dev
```
### Folder data structure
    .
    ├── src                 
        ├── assets          
        ├── boot            # extern libraries
        ├── components      # custom components
        ├── css             # styles
        ├── layouts         # app layouts
        ├── pages           
        ├── router          
        ├── statics
        ├── store        
        ├── App.vue         # Application entry point
        ├── index.template.html        
        └── README.md
